HTML Structure

An HTML document is made of different parts:

<!DOCTYPE html> : Declares the document type and version of HTML.
<html> </html> : Root element that contains the entire HTML document.
<head> </head> : Contains meta information like the title and linked CSS.
<title> </title> : Sets the title of the webpage (seen in the browser tab).
<body> </body> : Contains everything you want to display on the webpage like text, images, and links.

HTML Comments

Comments in HTML help developers explain their code.
They are not shown on the webpage.
Example of a comment:
<!-- This is a comment -->
